# Linked List Implementation (Java)

## Objective
Implement a custom singly linked list from scratch without using built-in data structures.

## Overview
This project focuses on understanding how linked lists work internally by building one using nodes and references.

## Key Concepts
- Nodes (data + reference to next node)
- Traversal
- Dynamic memory structure
- Pointer/reference management

## Features Implemented
- Add item to end of list
- Add item to front of list
- Remove item by ID
- Search for item
- Update item quantity

## Example Structure
Each node contains:
- Item object
- Reference to next node

## Challenges
- Handling edge cases (empty list, removing head)
- Fixing pointer issues when removing nodes
- Avoiding null pointer errors

## What I Learned
- How linked lists differ from arrays
- How references work in Java
- How to debug logical errors in data structures

## Future Improvements
- Convert to doubly linked list
- Add sorting functionality
- Improve efficiency of search
